## What is this ?

Btc seeder is a simple web app to help you convert a mnemonic passphrase to
different standards.

## Supported standards:
* [BIP39](https://github.com/bitcoin/bips/blob/master/bip-0039.mediawiki)
* [Mnemonic.js](https://github.com/ggozad/mnemonic.js)

## Installation

To install, run

`npm install`

`grunt bowercopy`
